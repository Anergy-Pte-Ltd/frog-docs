Proposed TURBO Applications¶

Purpose

The specification has been created to to provide a high level detail for the user application interface for TURBO.

Author

Matthew Martella

Date

13/02/2021

Revision

A

topic

Sepcification for Project TURBO-FROG for the implementation of the highest level menu in TURBO

Goto Apps Collection or Apps Collection or Admin

Introduction¶

TURBO being an implementation of the Odoo platform relies upon the notion of ‘apps’. In the technical implementation they are like a level 0 menu item. The level 1 menu items are then the ones you see across the top of the screen, level 2 in their dropdowns, level 3 in side pop-outs etc.

This document provides a framework for what the intended apps at this level shall be. This should be updated in accordance with any changes made.
