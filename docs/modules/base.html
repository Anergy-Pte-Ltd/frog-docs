Base Module¶

Overview¶

The base module is the root for all other modules. It even lives in a separate part of the Odoo directory structure. The description for the module is:

The kernel of Odoo, needed for all installations.

Probably the more interesting question when reviewing this module is what does it actually feed into the other modules. This is useful for building a picture of how TURBO can take the greatest advantage of Odoo.
